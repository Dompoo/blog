- Remote Procedure Call, **원격 프로그램의 함수를 마치 로컬 함수처럼 호출할 수 있는 패턴이다.**
- 즉, 분산 네트워크 통신의 복잡성을 추상화하는 것이다.
- MSA의 통신방식으로 가끔 사용된다. (REST API가 주로 사용)
- 동작 방식
	- 클라이언트가 원격 함수를 실행
	- 요청을 직렬화 하여 전달
	- 요청을 역직렬화하여 실제 함수 실행
	- 실행 결과를 다시 직렬화하여 반환
- 주요 특징
	- 보통 동기적으로 작동
	- 서로 다른 언어간 통신 가능
	- 타입 안정성 제공
- `gRPC`가 주요 구현체이다.
