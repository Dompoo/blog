- **인메모리 데이터 저장소**이다.
- 모든 데이터를 메모리에 저장하기 때문에 **매우 빠르다.**
- Key-Value의 `Map`과 같은 구조를 기본으로 하되, 다양한 자료구조가 존재한다.
- 단일 스레드라서 원자성이 보장된다.
- 빠르고 가볍다는 장점 + 여러가지 확장에 의하여 활용 방안이 무궁무진하다.
### 지원하는 자료구조
- String : 숫자도 문자열로 저장되고, 내부적으로 이를 인식한다.
- List
- Set
- Sorted Set
- Hash
- ...
### 사용 사례
- DB 조회 결과 캐싱
- 분산 세션 스토어
- 실시간 카운터
- 분산 락
- 메시지큐
- ...
